# **App Name**: Lons

## Core Features:

- Message Submission: Users can post new text messages, optionally attaching images, and choose to post anonymously or with a provided username auto send to admin.
- Image Attachment: Functionality to upload and associate images with messages, making them visible in the feed.
- Anonymous Toggle: A clear user interface control allowing users to switch between posting with a chosen name and posting without any identity.
- Admin Page for Anonymous Chat: An administrative interface allowing the admin to view direct anonymous chats.

## Style Guidelines:

- Primary color: Muted yet discernible purple (#E1BDFF) to stand out against a dark theme and evoke creativity and anonymity.
- Background color: Deep, desaturated purple-gray (#2C2630) providing a serene dark mode experience, enhancing readability.
- Accent color: Vibrant blue-violet (#4F2CDB) to highlight interactive elements and call-to-action buttons, offering strong contrast.
- Headline font: 'Space Grotesk' (sans-serif) for a modern, slightly techy feel, suitable for post titles or important messages. Body font: 'Inter' (sans-serif) for high readability in longer text blocks like message content.
- Subtle and modern vector icons to denote message actions, attachments, and user interactions, blending seamlessly with the dark theme.
- Clean, card-based layout for messages in the feed, ensuring easy scanning and readability on various screen sizes, with ample whitespace.
- Smooth fade-in or slide-up animations for new messages appearing in the feed and subtle transitions for button states.
