# **App Name**: VaultFeedback

## Core Features:

- Anonymous Text & Image Submission: Users can compose detailed text messages and attach images, then securely submit their feedback.
- User Identity Toggle: A prominent and intuitive toggle allows users to choose between anonymous submission and submitting with their identified username.
- Clear Submission Confirmation: After successfully submitting feedback, users receive an immediate, clear visual confirmation message.
- Admin Feedback Dashboard: A secure, administrative interface to view, filter, and manage all submitted feedback, including text and attached images.
- AI Content Summarization & Categorization Tool: An AI-powered tool to automatically summarize longer feedback messages and suggest categories (e.g., 'bug', 'feature request', 'general inquiry') to assist admins in quick triage.

## Style Guidelines:

- Color scheme: Dark theme, invoking professionalism and security. Primary actions and important elements use a deep, authoritative purple (#6C59B3).
- Background color: A subtly dark, desaturated charcoal with a hint of purple for depth and focus (#222126).
- Accent color: A vibrant yet clean blue (#5C7CFA) is used for interactive elements, highlights, and status indicators, providing strong contrast and visual interest.
- Body and headline font: 'Inter', a clean, modern, sans-serif typeface that ensures readability and a contemporary feel across the application.
- Utilize minimalist line icons that align with the clean design. Icons should clearly represent actions like message, attachment, user profile, and anonymity.
- Embrace a clean, structured layout with ample negative space to enhance clarity and focus, particularly for feedback submission forms and the admin dashboard. Implement a grid-based approach for consistency.
- Subtle, non-distracting animations for submission confirmations and state changes, like toggling identity. Animations should feel responsive and enhance user interaction without overwhelming it.
